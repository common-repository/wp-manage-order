Manage order for posts or not hierarchical Custom types

== Description ==

This plugin let you manage order for not hierarchical (custom) post types

== Installation ==

1. Upload folder wp-manage-order to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin from settings menu
